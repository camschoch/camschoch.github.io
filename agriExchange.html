Agri-Exchange



AgriExchange is a group project in which we set out to build an application for urban gardeners to get gardening tips, display on a map all the local farmers markets, and see the weekly forecast to determine if there will be any days that planting would be ill advised. On the project, I was responsible for working with the APIs and building the backend with them. We implemented a farmer’s market API to give the address of all the local farmers markets in the current logged in user’s zip code along with a weather API in order to get the forecast for the week and alert the user if there will be an excessive amount of rain or frost, as well as a google Maps Geo location API to convert addresses to Geo coordinates so we could accurately give information to the users position as well as display the nearby farmers markets on a map for the user.
